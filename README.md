## 🎯 Objectif

Étudier comment le Deep Learning **adapte ses architectures à la structure des données**
(tabulaire, image, séquentielle), en reliant systématiquement **théorie**, **implémentation PyTorch**
et **expérimentation rigoureuse**.

---

## 🧠 Contenu du projet

| Partie | Architecture | Jeu de données | Tâche | Résultat |
|:------:|--------------|----------------|-------|:--------:|
| **I**   | MLP | Breast Cancer Wisconsin | Classification tabulaire | **97.6 %** accuracy |
| **II**  | CNN (LeNet) | CIFAR-10 | Classification d'images | **84.7 %** accuracy |
| **III** | RNN / LSTM / GRU / Seq2Seq | fra-eng (Tatoeba) | Traduction neuronale | **BLEU 31.4** |
